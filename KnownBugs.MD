# 🐛 Known Bugs

- Mission Skipping: Some missions seem to be skipped over. We’re aware of this issue and a fix will be included in the next version.
- Personnel Requirement: There’s an issue where the personnel requirement is not being fulfilled. We’re currently testing a workaround for this.
- Vehicle Mapping: Vehicles are not appearing on the map when they should be. This issue has been addressed in version 1.0.2.
- EMS Chief Response: There was an issue where the EMS Chief was not responding as expected. This has been fixed in version 1.0.2.
- Personnel Requirement Dispatch: Vehicles are not being dispatched for personnel requirements. This issue is currently a work in progress.
- Duplicate Dispatches: There was an issue causing duplicate dispatches. This has been fixed in version 1.0.2.

## We appreciate your patience as we work to resolve these issues. Your feedback helps us improve! 🙌
